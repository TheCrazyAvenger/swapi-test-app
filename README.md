# Swapi App

React-Native application

## Installation

1. Create .env file with BASE_URL:

- BASE_URL=SWAPI_KEY

2. Android:

```bash
yarn install
```

3. iOS:

```bash
yarn install
cd ios
pod install
```

## Running

1. Start packager:

```bash
yarn start
```

2. Android:

```bash
yarn run-android
```

3. iOS:

```bash
yarn run-ios
```
